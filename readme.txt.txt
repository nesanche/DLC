INSTRUCCIONES PARA CORRER EL PROGRAMA:

1) Descargarse la base de datos de esta URL:

2) Descargar la carpeta DocumentosTP1 para correrlo localmente (en caso de querer usar el indexador).

3) Ingresar a la carpeta src/java, buscar el archivo config.properties y editar las dos propiedades:
	a) En database_folder colocar la ruta absoluta a la carpeta con la base de datos (asegurarse que su nombre es dlc_database.db). El path debe estar en el 	formato Disco:\\directorio1\\directorio2\\directorio3 (sin el nombre del archivo, sólo la carpeta).
	b) En documents_folder colocar la ruta absoluta de la carpeta DocumentosTP1 con el mismo formato que la ruta anterior.

4) Abrir el proyecto con el IDE Netbeans y correrlo en glassfish.

5) Comenzar a buscar e indexar :) 

Troubleshooting: Puede pasar que cuando se importa el proyecto haya que cambiar la referencia al sqlitejdbc-v056.jar, en ese caso, hacer el cambio pertinente.
